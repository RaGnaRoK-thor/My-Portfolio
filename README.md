# Ushnish Chowdhury — Portfolio

A visually stunning personal portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

## Features

- ⚡ Snap-scrolling single-page layout
- 🎨 Glassmorphism & gradient design
- 🎬 Rich Framer Motion animations
- 📱 Fully responsive with mobile nav
- 🖼️ Custom SVG "UC" logo

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the site.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

| Layer     | Technology           |
| --------- | -------------------- |
| Framework | React 18             |
| Bundler   | Vite 6               |
| Styling   | Tailwind CSS 3       |
| Animation | Framer Motion 11     |
| Icons     | React Icons          |
